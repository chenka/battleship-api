# battleship-api
For interview

## Prerequisites
Node.js

MongoDB

## Installing
```
npm install
npm start
```

## API
`POST /attack/:gameId`

`POST /deploy/:gameId`

`POST /games`

`POST /attack/:gameId`


## User Interface
You can view game result at `/grids/:gameID


## Running the simulation
You have to start app by `npm start` before simulate.
Open new terminal and run `npm run simulate`



## Running the tests
`npm test`
